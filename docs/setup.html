🪑 Hotdesk Auto-Booker

Automatyczne rezerwowanie biurka codziennie o 00:01

⏱️ Konfiguracja zajmie około 5-10 minut

1

Załóż konto GitHub (jeśli nie masz)

GitHub to darmowa platforma, która będzie automatycznie rezerwować Twoje biurko każdego dnia.

Załóż konto GitHub →

Masz już konto? Przejdź do kroku 2.

2

Stwórz swoją kopię auto-bookera

Kliknij przycisk poniżej, aby utworzyć własną kopię systemu rezerwacji:

Utwórz moją kopię →
Ważne: Na następnej stronie:
  • Wpisz dowolną nazwę (np. moj-hotdesk)
  • Wybierz "Private" (prywatne)
  • Kliknij "Create repository"
3

Pobierz swój token z Hotdesk

Najpierw otwórz Hotdesk i zaloguj się:

Otwórz Hotdesk →

Następnie przeciągnij ten przycisk na pasek zakładek:

📋 Kopiuj Token

Jak przeciągnąć?
  1. Pokaż pasek zakładek: Ctrl+Shift+B (Windows) lub Cmd+Shift+B (Mac)
  2. Przeciągnij fioletowy przycisk "📋 Kopiuj Token" na pasek zakładek
  3. Będąc na stronie Hotdesk, kliknij zakładkę "📋 Kopiuj Token"
4

Dodaj token do GitHub

Otwórz ustawienia swojego repozytorium:

  1. Wejdź na github.com
  2. Kliknij w swoje nowe repozytorium (np. moj-hotdesk)
  3. Kliknij Settings (Ustawienia) → Secrets and variables → Actions
  4. Kliknij "New repository secret"
  5. Name: HOTDESK_REFRESH_TOKEN
  6. Secret: wklej token (Ctrl+V / Cmd+V)
  7. Kliknij "Add secret"
✅ Token jest bezpiecznie zaszyfrowany i widoczny tylko dla Ciebie.
5

Ustaw swoje preferencje

Edytuj plik z ustawieniami, aby wybrać preferowane biurka:

  1. W swoim repozytorium otwórz plik: .github/workflows/daily-booking.yml
  2. Kliknij ikonę ołówka (✏️) aby edytować
  3. Znajdź sekcję env: i zmień:
env: PREFERRED_DESKS: S05,S15,S10,S14 # Twoje ulubione biurka BOOKING_SUBJECT: "Twoje Imię" # Opis rezerwacji

Dostępne biurka: S01, S02, S03... S18
Wpisz je w kolejności od najbardziej do najmniej preferowanego.

  1. Kliknij "Commit changes" aby zapisać
6

Przetestuj!

Sprawdź czy wszystko działa:

  1. W swoim repozytorium kliknij zakładkę "Actions"
  2. Kliknij "Daily Hotdesk Booking" po lewej
  3. Kliknij "Run workflow" → "Run workflow"
  4. Poczekaj ~30 sekund i odśwież stronę
  5. Kliknij na uruchomione zadanie aby zobaczyć logi
🎉 Gotowe! Od teraz biurko będzie rezerwowane automatycznie codziennie o 00:01.
?

Pytania i problemy

Token wygasł (błąd 404)

Zaloguj się ponownie do Hotdesk, użyj bookmarkletu "📋 Kopiuj Token" i zaktualizuj secret w GitHub.

Nie widzę zakładki Actions

Upewnij się, że repozytorium jest Twoje (nie oryginalne timolszewski/hotdesk-booker).

Chcę zmienić preferowane biurka

Edytuj plik .github/workflows/daily-booking.yml w swoim repozytorium.
